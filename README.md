# 📊 ETL com Python - Manipulação de Dados

Este projeto foi desenvolvido com base na mentoria da DIO (Digital Innovation One):

**ETL com Python - Como Manipular Grandes Massas de Dados com Python**

## 🎥 Vídeo da Mentoria

Confira o conteúdo que serviu de base para o projeto:  
https://www.youtube.com/watch?v=R0rMMDOuXEQ

## 🚀 Objetivo

O objetivo deste projeto é demonstrar, na prática, o processo de ETL (Extract, Transform, Load), aplicando técnicas para:

- Extrair dados de múltiplas planilhas Excel  
- Tratar e transformar os dados  
- Consolidar as informações em uma única base  
- Gerar uma base limpa e estruturada para análise  

## 🧠 Conceitos Aplicados

- ETL (Extract, Transform, Load)  
- Manipulação de dados com Python  
- Leitura e escrita de arquivos Excel  
- Limpeza e padronização de dados  
- Consolidação de múltiplas fontes  

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- OpenPyXL
